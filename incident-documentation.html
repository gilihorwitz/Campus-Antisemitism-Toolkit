Tips for Documentation

  • Document immediately: Write down details while they're fresh in your memory.
  • Be factual: Stick to what happened, not assumptions about intent.
  • Save evidence: Take photos, screenshots, or save messages immediately.
  • Keep copies: Store documentation in multiple places (cloud, email to yourself).
  • Include context: Note if this is part of a pattern or related to other incidents.
  • Get witness contact info: Ask witnesses if they're willing to provide statements.

When to Use This Template

Use this template for any incident that makes you feel targeted, unsafe, or discriminated against because of your Jewish identity. Even if you're not sure whether something "counts," document it. You can decide later whether to report it.
